# Vita — Demo Web (v3)

Esta versión copia **más de cerca** el look & flow de tu app (según capturas):
- Home estilo “Hola, Favio…” + tarjetas grandes (Evaluación / Chequeo emocional)
- Triage con header azul + barra de progreso
- Paso 1: datos básicos + antecedentes (con “ver más”)
- Paso 2: lista de síntomas con íconos/alerta
- Paso 3: zona del dolor + tipo de dolor
- Paso 4: signos vitales por rangos + botón **Calcular**
- Resultado con: color + confianza + plan inmediato + recomendaciones + botones inferiores

## Correr
```bash
npm install
npm run dev
```
Abrir: http://localhost:3000

## Ruta jurado
Inicio → Iniciar evaluación → (4 pasos) → Resultado → Portal profesional / Resumen.
